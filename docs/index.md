# Welcome to the PlatON Documentation Site！


本文档旨在帮助不同的用户从不同角度了解PlatON，并使用PlatON。它涵盖了PlatON的基础介绍，各种方案设计的介绍，以及关于合约和DAPP的高级使用说明。


如果你是初次开始了解如何使用PlatON，我们建议你按照以下路径开启你的学习之旅~
